# Strombit — Biodata Web Developer Fullstack

Halo — saya Strombit. Saya seorang Web Developer Fullstack yang berfokus pada pembuatan aplikasi web yang bersih, cepat, dan mudah dipelihara. Saya menyukai pemecahan masalah nyata dengan solusi teknis yang pragmatis dan desain antarmuka yang ramah pengguna.

## Ringkasan
- Berpengalaman membangun aplikasi end-to-end: desain UI, frontend modern, backend API, hingga deployment.
- Pendekatan berorientasi pengguna dan performa: mengutamakan UX, optimisasi, dan skalabilitas.
- Terbiasa bekerja dalam tim lintas-fungsi dengan praktik terbaik (code review, testing, CI/CD).

## Keahlian Utama
- Frontend: HTML5, CSS3, JavaScript (ES6+), TypeScript, React, Vue, SASS, responsive design, a11y.
- Backend: Node.js (Express/Koa), Python (Flask/Django), RESTful API, GraphQL.
- Database & Cache: PostgreSQL, MySQL, MongoDB, Redis.
- DevOps & Tooling: Docker, Git, GitHub, GitHub Actions, Nginx, deployment ke cloud (DigitalOcean, AWS/GCP dasar).
- Testing & Kualitas: Jest, React Testing Library, Mocha, Supertest, linting dan format otomatis.

## Proyek Unggulan
- Aplikasi e‑commerce fullstack: katalog produk, keranjang, checkout, integrasi pembayaran, dashboard admin.
- CMS ringkas: editor WYSIWYG, manajemen konten dan pengguna, versioning.
- Layanan microservice untuk pemrosesan background: worker + Redis untuk antrian tugas.

Semua proyek didesain agar mudah direview — struktur jelas, dokumentasi ringkas, dan skrip untuk setup lokal.

## Cara Bekerja
- Pecah masalah menjadi tugas kecil dan prioritaskan yang berdampak tinggi.
- Tulis kode yang dapat dibaca, diuji, dan mudah dipelihara; gunakan linting dan format yang konsisten.
- Komunikasi terbuka: jelaskan konteks pada PR dan terima masukan konstruktif.

## Pendidikan & Pengembangan
- Pembelajaran berkelanjutan melalui kursus, dokumentasi resmi, dan kontribusi open source.
- Aktif mengikuti komunitas developer untuk memperbarui praktik dan alat yang digunakan.
